# 选课小本本制作指南

## 0、所需材料
1. 问卷星导出的excel文件
2. 声明.pdf

## 1、程序运行
运行[JudgeQu.py](https://www.baidu.com)程序，得到**选课小本本.docx**文件。

![](README_pic\目录.png)

## 2、docx2PDF
得到docx文件后，需要做以下事情：

### 1. 制作目录
在第一页插入目录，目录插入方式如图所示：

![](README_pic\插入目录.png)
### 2. 导出PDF
按照说明导出PDF，建议添加边栏书签

![](README_pic\导出PDF.png)

## 3、合并PDF
可使用在线合并PDF或者本地合并功能，合并**声明.pdf**以及**之前导出的PDF**，得到最终的版本。
